# Digispace

Demo interattiva Phygispace / Digispace per il team.

## Pubblicazione

Il sito è progettato come pagina statica autonoma. Il file principale deve essere `index.html` nella root del repository.

Dopo il caricamento di `index.html`, attivare GitHub Pages da **Settings → Pages → Build and deployment → Deploy from a branch → main / root**.
